## 🎮 How to Play

- Use the **arrow keys** (↑ ↓ ← →) to control the snake’s movement.  
- Eat the **green apples** to grow longer and increase your score.  
- Avoid running into the **walls** or into the **snake’s own body** — doing so ends the game.
